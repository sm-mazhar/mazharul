Active PhD Research · University of North Texas

Gamification of ADHD Treatment
An accessibility-centered HCI research direction supported by adaptive NLP and ML

I am developing a research-driven framework for gamified ADHD treatment that prioritizes accessible interaction, cognitive support, and meaningful user engagement. The work investigates how HCI methods and accessibility principles can shape a game-like therapeutic experience, while NLP and machine learning help personalize feedback, pacing, and adaptation in a careful, human-centered way.

Secondary area: NLP Secondary area: Machine Learning Supporting area: Digital Health Supporting area: Ethical AI

Accessible Game Mechanics

Interaction patterns are designed to be supportive, motivating, and cognitively manageable for people with ADHD.

NLP Support Layer

NLP helps personalize prompts, responses, and communication features so the experience can adapt to users more naturally.

Accessibility-First HCI

The core research contribution is an inclusive interface strategy built around attention, comprehension, and reduced cognitive overload.

Ethical & Fair AI

Developed with strong emphasis on Fairness, Bias Mitigation, and Privacy, ensuring equitable treatment for all users regardless of background.

Multimodal Fusion

Behavioral and interaction signals can inform adaptation without making the interface feel overwhelming or overly complex.

Personalized Outcomes

Systematic application of research tenacity to address complex, open-ended problems in adaptive systems and digital health innovation.

Research Projects

A selection of research projects spanning accessibility, NLP, biomedical informatics, and intelligent systems.

Current PhD

Gamification of ADHD Treatment

Accessibility-centered HCI research exploring supportive game mechanics, with NLP and ML used to personalize therapeutic experiences.

This project investigates how game-based interaction can make ADHD care more engaging, structured, and cognitively supportive for users with different accessibility needs. The work combines inclusive HCI design with adaptive feedback systems so the experience can respond to attention, motivation, and task progression more intelligently. NLP and machine learning act as supporting layers for personalization, communication, and therapeutic guidance rather than replacing the human-centered design core.

Wikipedia-Empowered NLI for Web Search

Natural language interface leveraging Wikipedia as structured knowledge to improve query understanding and information retrieval.

This work develops a natural language interface that uses Wikipedia as a structured knowledge layer for clarifying entities, concepts, and user intent during web search. The project focuses on improving how ambiguous queries are interpreted so search becomes more context-aware, semantically grounded, and useful for real users. It connects NLP, knowledge representation, and information retrieval in a way that supports more explainable and human-friendly search interaction.

Biomedical Knowledge Discovery

MetaMap and text mining on MEDLINE to uncover hidden relationships between medical concepts for clinical decision support.

This research theme uses biomedical text mining and domain-aware concept mapping to uncover meaningful relationships hidden across large MEDLINE-scale literature collections. One direction emphasizes domain knowledge integration to discover previously overlooked links between medical concepts, while another leverages MetaMap and text mining for relationship extraction. Together, these studies support smarter biomedical knowledge discovery workflows for literature analysis, evidence synthesis, and future decision support systems.

Gesture-Based Smart Wheelchair

IoT-enabled wheelchair controlled by hand gestures with emergency SMS functionality - intersects directly with current accessibility research.

This assistive system was designed to improve mobility support through hand-gesture control, giving users a more intuitive interaction alternative to conventional physical input methods. The wheelchair also integrates emergency SMS functionality so caregivers or family members can be contacted quickly when urgent support is needed. The project connects directly to accessibility research because it addresses autonomy, usability, and assistive interaction design in a real-world context.
Related publication: Gesture-based Smart Wheelchair for Assisting Physically Challenged People

Sentiment Analysis Systems

Multiple projects analyzing Twitter and review data using hybrid supervised/unsupervised NLP and lexicon-based deep learning methods.

This line of work focused on identifying emotional tone, opinion polarity, and human sentiment patterns from noisy real-world text such as tweets and review data. The project explored hybrid NLP approaches that combine lexicon-based analysis with supervised and unsupervised learning strategies for stronger sentiment classification. It helped build a strong foundation in applied NLP, text analytics, and feature-driven modeling that later informed broader machine learning research directions.
Related publication: Sentiment Analysis of Tweet Data: The Study of Sentimental State of Human from Tweet Text

Multi-Mode IoT Robot

Arduino-based robot controllable via voice, wireless, and full-automatic modes with IoT long-distance communication.

This project explored a flexible robotics platform built to support multiple interaction modes depending on user needs, environmental context, and control preferences. The robot could operate through wireless control, voice recognition, and fully automatic behavior while staying connected through IoT-based long-distance communication. It brought together embedded systems, sensing, control logic, and interface adaptability in a way that emphasized both technical robustness and practical usability.
